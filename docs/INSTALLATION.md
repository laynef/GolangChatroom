# Installation

## Requirements

- PostgreSQL 14
- Node version >= 16
- Yarn >= 1.22.17
- Go >= 1.17.5

## Setup

1. Create a new database and run the server
2. Add the database details into a .env file. An example file is provided
3. `$ go install`
5. `$ yarn install`
6. `yarn build`
7. `air`
8. Listen to port 3000
