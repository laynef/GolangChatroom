# Installation

## Requirements

- PostgreSQL 14
- Node version >= 16
- Go >= 1.17.5

## Setup

1. Create a new database and run the server
2. Add the database details into a .env file. An example file is provided
3. `$ go install`
4. `$ go build`
5. `$ yarn install`
6. `yarn build`
7. `go run main.go`
